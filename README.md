# 水下机器人ROV电力载波&12V电源集成板

用于ROV或AUV下位机的电力载波解码与12V供电转5V供电板

**大小：150 x 19 mm**

电力载波解码模块使用通用的高通主控电力载波模块或东软电力载波模块，只要**接口定义一致就可以使用**

已加入M3螺柱定位孔

## V2.0版本

改为四层板设计，提高数字部分稳定性

使用了规格统一的供电模块用于24V-12V-5V的转换

单独引出电力载波信号接口，支持12V-36V电压的电力载波信号直接输入

优化网口布线

优化布局，现在可以支持叠层放置树莓派或其他单板计算机

该方案理论上可以单独实现一个基于树莓派和电力载波的水下机器人控制仓，但由于现有设备需求暂时不会应用
